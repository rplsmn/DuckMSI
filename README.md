# DuckDB WASM Web App

A web application for uploading Parquet files and running SQL queries using DuckDB WASM.

## Quick Start

```bash
# Development
npm install
npm run dev

# Deploy to GitHub Pages
./deploy.sh
```

## Core Files

Add new features by editing:

- **`index.html`** - Main UI structure and styling
- **`src/main.js`** - DuckDB initialization and database operations
- **`src/app.js`** - File upload handling and query execution
